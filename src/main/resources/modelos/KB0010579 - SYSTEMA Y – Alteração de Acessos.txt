 Revisão Semestral de Acessos
Regularmente os acessos ao Sistema deverão ser revisados pelo Gestor do time de GQ-Digital.
A lista de acessos ativos no sistema será gerada pelo time de Digital - Acessos

Observação: A equipe de Gestão de Acessos utiliza apenas o módulo “Configurar Segurança”. Os módulos “Criar nova conexão” e “Editar esta conexão” são utilizados somente pelos administradores do sistema.

Users: Utilizado para criar usuário;
Groups: Utilizado para criar grupo;

Roles: Utilizado para criar regra;
